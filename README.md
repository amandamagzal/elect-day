# Election Day Management System

*Design & Implementation of Information Systems Course Project.*

This system manages the operational election day flow: voters arriving, voting process itself, parties, vote recording, etc.  
Architecture used the **Boundary – Control – Entity (BCE)** model.

- Java application
- Database: Microsoft Access
- Reporting: JasperReports (reports can be generated for branch #2)
